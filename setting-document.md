# 金笼 · 设定文档

> 本文档整合现有设计稿，作为所有任务线的设定基准。
> 
> **一致性约束**：荷官×赌神线已完成全部任务文本，其他线如需调整须单独讨论。

---

## 一、世界观及核心冲突

### 1.1 时代与背景

- **时代**：架空民国，军阀混战末期
- **金笼**：表面是地下赌场，实际是革命地下党的秘密据点。赌场生意既是掩护也是资金来源——用赌场利润购买军火、资助前线、转运情报
- **地理**：建在某城废弃的官府地窖中，五个通用区域 + 三个男主专属区域

### 1.2 核心冲突：沈万钧 vs 金笼

| 要素 | 内容 |
|---|---|
| **沈万钧** | 黑市军火商，原金笼军火供货商。陆鹤声发现沈暗中向军阀倒卖武器、出卖地下党安全屋位置后，切断合作并销毁交易证据 |
| **沈的动机** | 金笼密室中存有沈参与军火交易的完整账册——一旦曝光，沈将被革命组织和军阀两方追杀。沈必须夺回账册，顺便报复陆鹤声 |
| **账册位置** | **密室保险箱**（陆老板的核心机密存放处），非守门人金库 |

### 1.3 沈的三步计划

```
第一步 · 锦标赛（调虎离山）
  └─ 安排内奸将金筹码放入奖池，抬高赛事规格
  └─ 大量陌生面孔涌入赌场，掩护潜入
  └─ 安保注意力集中在前台

第二步 · 炸弹（制造混乱）
  └─ 安放在地下层，绑在严九身上
  └─ 目的：制造恐慌，逼出密室位置
  └─ 荷官被利用调整了排班，为炸弹安放创造窗口

第三步 · 围剿（总攻）
  └─ 沈万钧亲自坐镇
  └─ 里应外合攻入赌场
  └─ 目标：活捉陆鹤声、夺取密室账册
```

### 1.4 金筹码系统

金笼初建时铸了一批金筹码作为创始凭证，不在赌桌流通。女主需集齐 5 枚才能离开副本。

| # | 获取方式 | 时间段 | 来源 |
|---|---|---|---|
| ① | 探索 | 第一幕 | 身份危机任务的必经场所中 |
| ② | 主线 | 第一幕末 | 从赌场元老吴半城手中获得 |
| ③ | 主线 | 第二幕·锦标赛 | 锦标赛奖池中 |
| ④ | 探索 | 第二幕·炸弹 | 绑在人质严九身上 |
| ⑤ | 主线 | 第三幕·围剿 | 密室保险箱中 |

5 枚未集齐 → **困局**（失败结局）。

## 二、赌场区域设定

### 2.1 通用区域（所有身份组合可进入）

| 区域 | 描述 | 功能 |
|---|---|---|
| **大厅** | 赌场主体，烟雾缭绕，赌桌林立，水晶灯照明 | 赌局、社交、信息收集、锦标赛主场地 |
| **绸缎阁** | VIP 包厢区，单面玻璃，水晶灯，私密空间 | 情报接头处、高层会面、老板观战点 |
| **账房** | 财务+情报中心，吴半城坐镇 | 账目查证、金筹码②获取、权力结构调查 |
| **地下层** | 走私通道/囚室/军火库，潮湿砖墙，滴水声 | 炸弹事件主场地、严九牢房、金筹码④ |
| **后台** | 走廊、扣押室、杂物间、员工区 | 潜入路线、工具存放、私下会面 |

### 2.2 男主专属区域（仅对应男主线可进入）

| 男主线 | 专属场景 | 描述 | 解锁时机 | 绑定任务 |
|---|---|---|---|---|
| **荷官** | 钟楼天台 | 赌场附近废弃钟楼顶部，可俯瞰整个街区。藏着荷官的逃跑包和情报死信箱 | 第一幕末·一杯茶后 | 天台教学、天台重逢、围剿前夜 |
| **老板** | 密室·书房 | 赌场最深处的私人空间，红木书架、地图、密电码本。墙上挂着一张旧合照——年轻时的老板和战友们 | 第二幕·老板邀请/潜入 | 真相揭露、围剿前夜、最终抉择 |
| **打手** | 地下斗兽场 | 地下非法格斗场，血迹斑斑的沙地，铁笼围栏。更深处连接严九牢房 | 第二幕·打手线任务 | 格斗赛、发现师父、围剿前夜 |

### 2.3 区域解锁规则

- 通用区域中，部分区域需任务进度解锁（如地下层在炸弹事件时解锁、密室在围剿时解锁）
- 男主专属区域由剧情自然触发，仅在对应男主线中出现
- 不同女主身份在通用区域的活动权限不同（杀手需伪装、千金需买通才能进入某些区域）

## 三、男女主身份设定

### 3.1 三种男主身份与背景故事

#### 荷官

| 项目 | 内容 |
|---|---|
| **表面身份** | 金笼首席荷官 |
| **真实身份** | 地下党情报员，利用牌桌套取各方情报 |
| **秘密** | 直属上级（安保副管事）是沈万钧的内应。上级知道荷官的情报员身份，以此要挟他配合"清场转移物资"。荷官被迫调整了锦标赛夜的地下层排班，客观上为炸弹安放创造了窗口。他不是放炸弹的人，是被利用为炸弹创造了条件 |
| **留在守门人身边的理由** | **反间 + 安保监控**。守门人是沈安插在金笼的人，暗中调安保、替沈的人开路。荷官必须寸步不离地盯着他，掌握他什么时候动手（已在荷官×赌神线中确认） |
| **个人目标** | 保住情报网、摆脱沈的胁迫，不想背叛金笼但不能让身份曝光 |
| **帮女主的逻辑** | 女主打破了赌场既有格局，他想借她搅浑水给自己争取空间。后期变为真心 |
| **核心弧线** | 从"被所有人利用的棋子"到"主动做出选择的人" |

#### 老板（陆鹤声）

| 项目 | 内容 |
|---|---|
| **表面身份** | 金笼创建者，赌场老板 |
| **真实身份** | 地下党在本城的负责人，儒雅但手段果决 |
| **秘密** | 已察觉沈万钧在渗透金笼但不知具体计划。暗中布局反制，不敢打草惊蛇——因为沈可能掌握了部分地下党名单 |
| **个人目标** | 守住金笼据点，将沈的罪证（密室中的账册）送交上级组织 |
| **帮女主的逻辑** | 评估她的利用价值——外来者可以做自己人不方便做的事。利用在先，感情在后；面对失控会不安 |
| **核心弧线** | 从"掌控一切的棋手"到"愿意放手信任一个人" |

#### 打手

| 项目 | 内容 |
|---|---|
| **表面身份** | 金笼底层打手，负责看守地下层和催债 |
| **真实身份** | 军人出身，部队打散后流落至此，卖命还债 |
| **师父·严九** | 打手的军中老长官。部队打散后收留养育了他。严九后来被沈万钧收编，替沈管军火、收债。沈与金笼翻脸后，严九作为沈的心腹被陆鹤声关进地下层牢房数年。**打手不知道师父就关在他脚下**——沈的人用假信息骗他"师父在我们手上，好好干活他就平安" |
| **秘密** | 欠沈万钧手下高利贷（为师父借的）。沈的人胁迫他在围剿时从内部打开地下层的门——开门赌场的人会死，不开门沈的人就对师父动手 |
| **个人目标** | 还债、保师父平安。在金笼底层待久了，逐渐感受到这群人在做的事——替不认识的人担风险——与师父"只管自己人"的规矩不同 |
| **帮女主的逻辑** | 具体交换——女主帮他还债/解决沈的威胁，他为她做事。底层人的利益交换，过程中产生真感情 |
| **核心弧线** | 从"师父的影子"到"独立做判断的人"。与师父的裂缝不是好 vs 坏，是两套世界观——师父认为乱世只管自己人，打手开始觉得不认识的人也值得管 |

### 3.2 三种女主身份与背景故事

#### 赌神

| 项目 | 内容 |
|---|---|
| **一句话定位** | 江湖上成名的女赌手，靠实力吃饭 |
| **专属玩法** | **鹰眼**——观察系统。场景中出现放大镜标记，检查后获取隐藏信息（微表情、记号、线索） |
| **第一幕危机** | 守门人逼战——赌场常驻冠军当众逼她三局两胜，输了永久驱逐 |
| **核心能力** | 读牌、读人、读局。用观察力碾压对手 |
| **气质** | 冷静、自信、锋芒内敛。不动声色地掌控全局 |

#### 杀手

| 项目 | 内容 |
|---|---|
| **一句话定位** | 受雇潜入赌场的职业杀手 |
| **专属玩法** | **伪装**——换装系统。从衣架/储物柜取衣物进行乔装，进入不同区域。有时间窗口限制 |
| **第一幕危机** | 目击灭口——通过通风管潜入后，在地下层目击私刑，被发现后安保搜捕 |
| **核心能力** | 渗透、伪装、格斗。用隐蔽和暴力手段达成目标 |
| **气质** | 警觉、冷酷、高效。把每个环境当战场评估 |

#### 千金

| 项目 | 内容 |
|---|---|
| **一句话定位** | 名门千金，初入赌场的局外人 |
| **专属玩法** | **买通**——经济系统。随身首饰可变现，花钱从 NPC 手中购买信息/帮助/通行权。需权衡资金分配 |
| **第一幕危机** | 群狼环伺——保镖被骗走后独自面对连环围猎（扒手、放贷者、骗子） |
| **核心能力** | 社交、金钱、人脉。用资源和人际关系打开局面 |
| **气质** | 聪慧、大气、不谙江湖。用另一套规则理解这个世界 |

### 3.3 男女主初始关系

|  | 荷官 | 老板 | 打手 |
|---|---|---|---|
| **赌神** | 江湖上听过彼此名号，职业认同 | 听闻赌术名声，她的到来有利于撑场面 | 无，两个世界的人 |
| **杀手** | 完全陌生，互相试探 | 完全陌生，警觉又感兴趣 | 同为暴力谋生者，不需言语的理解 |
| **千金** | 只知道她家族名字 | 家族是地下党的秘密金主之一，既要保护又想利用 | 无，阶层差距大，不信任有钱人但被善意困惑 |

### 3.4 情感关系设计

**设计原则**：

- 男主的反应必须**符合身份逻辑**——他为什么会这样做，基于他的身份/秘密/目标
- 同一个男主在不同女主身份线中的反应可以不同，但**人设必须一致**
- 情感选择不是"好感度+/-"，而是**塑造"我是什么样的人、我和他是什么样的关系"**
- 3 个 S 级选择点决定结局路径，A 级选择增加风味和级联变化

**情感弧线对照**：

| 男主 | 情感发展逻辑 | 核心张力 |
|---|---|---|
| **荷官** | 从"互相利用"到"互相信任"。他被所有人利用，她是第一个不把他当工具的人 | 他想保护她但自身难保；她想帮他但不确定他是否值得信任 |
| **老板** | 从"利用与被利用"到"放下控制"。他习惯掌控一切，她是第一个让他失控的变量 | 他在利用和真心之间摇摆；她在合作和独立之间博弈 |
| **打手** | 从"利益交换"到"超越债务的牵挂"。底层人之间最朴素的互相帮衬 | 他被师父和道义两头撕扯；她帮他的动机他始终看不透 |

## 四、三线叙事拼图规划

### 4.1 设计理念

三个男主身份各揭示整体故事的一个侧面。玩家打完三条线后，拼出沈万钧 vs 金笼冲突的完整图景。

```
荷官线 → 揭示"沈的渗透手段"（内鬼网络、安保操控、情报战）
老板线 → 揭示"金笼的真相"（地下党据点、账册的意义、陆鹤声的信念与代价）
打手线 → 揭示"人的代价"（严九的忠诚与偏执、底层人被碾碎的命运、两套世界观的冲突）
```

### 4.2 各线视角与信息分配

#### 荷官线：谍战视角——"谁是内鬼？"

**玩家看到的**：
- 荷官在守门人身边的真实原因——反间+安保监控（非寻找账本）
- 守门人是沈安插在金笼的核心内鬼，操控安保部署
- 安保副管事（荷官的上级）也是沈的人，利用荷官的情报员身份要挟他配合
- 锦标赛夜排班被调整的真相——荷官被迫为之，不知是给炸弹创造窗口
- 围剿前荷官的情报网运作方式

**玩家不知道的**（其他线揭示）：
- 陆鹤声的完整身份和信念（老板线）
- 严九为什么被关、他和打手的关系（打手线）
- 沈万钧的完整动机和三步计划全貌（老板线拼合）

**荷官线的情感核心**：
他是被所有势力裹挟的棋子——沈的人要挟他，金笼的人怀疑他，他自己想保护情报网又不能暴露身份。女主是唯一一个看到他真实处境的人。

> **与现有荷官×赌神线的一致性**：
> - 守门人 = 沈的内鬼 ✓（任务10调查印证）
> - 荷官跟着守门人 = 反间+安保监控 ✓（任务10/12文本已修改）
> - 账本在密室 ✓（任务17"账本拿到了。陆老板已经带着它从密道撤离了"）
> - 严九暗示荷官效忠陆老板 ✓（任务14b"守门人的小跟班，什么时候变成陆老板的人了"）
> - 围剿前夜场景 = 钟楼天台 ✓

#### 老板线：全局视角——"这场仗到底在打什么？"

**玩家看到的**：
- 金笼的地下党据点真实身份——不是普通赌场，是革命事业的资金和情报枢纽
- 陆鹤声的身份：地下党本城负责人，守护据点、传递情报、保护战友
- 密室中账册的真正意义——不仅是沈的罪证，更是地下党安全网的关键一环
- 沈万钧的完整动机：夺回账册以自保，报复陆鹤声
- 陆鹤声为守护金笼做过的狠事（关押严九、牺牲个人情感、利用外来者）
- 墙上旧合照——年轻时的战友们，大部分已经不在了

**玩家不知道的**（其他线揭示）：
- 荷官的情报员身份和被胁迫的细节（荷官线）
- 底层打手和严九的个人恩怨（打手线）

**老板线的情感核心**：
他是掌控者，习惯把所有人当棋子——包括女主。他评估利用价值、安排信息传递、控制行动自由。但女主是第一个让他的棋局出现变数的人。他最难的不是打败沈万钧，而是放下控制、信任一个人。

> **需要确保的一致性**：
> - 老板线中荷官以配角出现时，行为逻辑需与荷官线人设一致
> - 密室位置、账本存放、围剿流程与荷官×赌神线一致
> - 守门人作为内鬼的设定跨线统一

#### 打手线：底层视角——"人被碾碎后还剩什么？"

**玩家看到的**：
- 严九的完整人物：军人出身→被沈收编→替沈做脏事→被陆关押→至今认沈为主
- 打手被欺骗的真相：师父不在沈手上，就关在他脚下
- 两套世界观的正面碰撞：师父"乱世只管自己人" vs 打手萌芽的"不认识的人也值得管"
- 底层人的命运——高利贷、卖命、被利用、没有选择权
- 围剿中打手面临的终极抉择：开门（师父活但金笼的人死）vs 不开门

**玩家不知道的**（其他线揭示）：
- 陆鹤声为什么关押严九（老板线——严九是沈的心腹）
- 荷官在安保调度中的角色（荷官线）
- 账册和金笼据点的政治意义（老板线）

**打手线的情感核心**：
他不关心政治、不懂情报战、不理解"为不认识的人担风险"。他只知道师父养大了他，现在师父要他帮沈。但在金笼底层待的时间够久后，他开始动摇——也许还有另一种活法。女主帮他还债/解决威胁的过程中，他第一次体验到"不是交易"的善意。

> **需要确保的一致性**：
> - 严九的人设（关押原因、与沈的关系）跨线统一
> - 炸弹绑在严九身上的设定与荷官线一致
> - 打手线中荷官/老板以配角出现时行为逻辑一致
